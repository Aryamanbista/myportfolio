My Project

Bhansaghar AR Menu App

Bhansaghar is an innovative Augmented Reality (AR) menu application designed for restaurants. It allows users to scan a physical menu and experience a dynamic AR menu, featuring interactive 3D food models. Each food item is brought to life with its dedicated 3D model and accompanied by unique theme music, enhancing the dining experience. The application was developed using Unity.

EcoBuddy

EcoBuddy is a community waste management web application designed to streamline and enhance waste collection processes. It provides a user-friendly platform for community members to schedule waste pickups, report issues, and view their pickup history. Built on the MERN stack (MongoDB, Express.js, React.js, and Node.js), the system also equips community administrators with tools to manage operations, view analytics, and generate reports, fostering more efficient and sustainable waste management.

Github

ClassClue

ClassClue is a native Android mobile application built to streamline academic scheduling for university students and administrators. The app provides a centralized platform where students can register for subjects, view events, and receive timely reminders for assignments and exams, with full offline access. A separate admin dashboard allows faculty to manage subjects and events seamlessly. Developed using Java with an MVVM architecture, it integrates Firebase for real-time data synchronization and authentication, and a local Room Database for offline functionality.
